* react 15.3
* webpack 2
* react-router 3
* react-redux

development
-----------
To run the code in your development environment:

1. Download and unpack
2. Run `npm install`
3. Start the development server for seed project with `npm run dev`
4. Point your browser to [http://localhost:2200](http://localhost:2200)
